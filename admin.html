How to connect your Google Sheet
  1. Go to sheets.google.com and create a new spreadsheet
  2. Name it: Viador Command Center
  3. Create 5 tabs at the bottom: KPIs | Pipeline | Marketing | Actuals | Notes
  4. Add headers in each tab:
    KPIs: Week | Deals Pipeline | Leads This Week | ViaScore Runs | LinkedIn Impressions
    Pipeline: Lead Name | Source | Loan Type | Amount | Stage | Next Action | Expected Close
    Marketing: Campaign | Channel | Spend | Leads | CPL | Closed | Revenue | ROI
    Actuals: Month | Actual Revenue | Projected Revenue | Variance
    Notes: Date | Note
  5. Click Share → Anyone with the link → Viewer
  6. Copy the Sheet ID from the URL: docs.google.com/spreadsheets/d/THIS_PART/edit
  7. Paste it in the "Connect Sheet" field at the top of the dashboard
  8. Click Connect — dashboard loads live data immediately

Manual setup required:

  1. Enable Google Sheets API on GCP project stone-index-490722-r2
  2. Create an API key restricted to Sheets API
  3. Add it as a Cloudflare Worker secret: wrangler secret put GOOGLE_SHEETS_API_KEY
  4. Deploy the updated worker: wrangler deploy
